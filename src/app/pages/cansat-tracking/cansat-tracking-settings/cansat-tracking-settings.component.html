How to get Chrome Serial Reader App Id

In your chrome browser click the 3 dot at top right -> More Tools -> Extensions. Locate Serial Reader and copy the ID and paste in the above box

If the chrome extension is not installed on your chrome, download app from github repo and add it. Since this app is not published in chrome app store you need to enable Developer Mode

App Installation Procedure - Go you to your extensions and click Load unpacked and locate the root of the downloaded app
